post /links

Create a new link from an url

  • Request

Body

Type: application/json

Schema:

{ "$schema": "http://json-schema.org/schema",
  "type": "object",
  "id": "<<baseUri>>",
  "properties": {
    "url": {
      "id": "<<baseUri>>/url",
      "type": "string"
    }
  },
  "required": ["url"]
}

Example:

{
  "url": "http://blog.octo.com/node-for-api/"
}

Retrieve the link representation you were looking for

  • Request
  • Response

URI Parameters

  • linkId: required (string)

HTTP status code 200

Body

Type: application/json

Schema:

{ "$schema": "http://json-schema.org/schema",
  "type": "object",
  "id": "<<baseUri>>",
  "properties": {
    "link": {
      "id": "<<baseUri>>/link",
      "type": "string" },
    "archive": {
      "id": "<<baseUri>>/archive",
      "type": "boolean" },
    "timestamp": {
      "id": "<<baseUri>>/timestamp",
      "type": "string"},
    "tags": {
      "id": "<<baseUri>>/tags",
      "type": "array",
      "items": [{
        "id": "<<baseUri>>/tags/0",
        "type": "string"},
        {"id": "<<baseUri>>/tags/1",
          "type": "string"}]},
    "comment": {
      "id": "<<baseUri>>/comment",
      "type": "string"}},
  "required": ["link","timestamp","archive","tags","comment"]
}

{
  "id": "adKFsdfdsuhnkj",

  "timestamp": "2015-02-16T09:25:43.511Z",
  "tags": [
    "rest",

  ],
