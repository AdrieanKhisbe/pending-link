    This method will do a partial update on an individual link

    get /links/{linkId}

    Retrieve link representation

    • Request
    • Response

    URI Parameters

    • linkId: required (string)

    HTTP status code 200

    Body

    Type: application/json

    Schema:

    { "$schema": "http://json-schema.org/schema",
  "type": "object",
  "id": "<<baseUri>>",
  "properties": {
    "link": {
      "id": "<<baseUri>>/link",
      "type": "string" },
    "archive": {
      "id": "<<baseUri>>/archive",
      "type": "boolean" },
    "timestamp": {
      "id": "<<baseUri>>/timestamp",
      "type": "string"},
    "tags": {
      "id": "<<baseUri>>/tags",
      "type": "array",
      "items": [{
        "id": "<<baseUri>>/tags/0",
        "type": "string"},
        {"id": "<<baseUri>>/tags/1",
          "type": "string"}]},
    "comment": {
      "id": "<<baseUri>>/comment",
      "type": "string"}},
  "required": ["link","timestamp","archive","tags","comment"]
}

    Example:

    { "link": "http://blog.octo.com/designer-une-api-rest/",
  "archive": true,
  "timestamp": "2015-02-16T09:25:43.511Z",
  "tags": ["rest", "api"],
  "comment": "Api Rocks"
}

    Type: application/xml
